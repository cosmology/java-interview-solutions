# Java Interview Solutions

This repo is a collection of some common [leetcode.com](https://www.leetcode.com) interview problems. It will expand into different areas over time.


## Pre-requisites
* [Java 21](https://www.oracle.com/java/technologies/downloads/#java21) use [SDKMAN](https://sdkman.io/)


## Quickstart

- [Build](#build)
- [Run individual class](#run-individual-class)
- [Run tests](#run-tests)
- [Run individual test](#run-individual-test)


## Build
  ``` sh
  ./gradlew build
  ```

## Run Individual Class
  ``` sh
  ./gradlew run -PmainClass=com.ivanprokic.leetcode.path.to.algorythm
  ```

## Run Tests
  ``` sh
  ./gradlew test
  ```

## Run Individual Test
  ``` sh
  ./gradlew test --tests SomeTestClass
  ```
